# Security Policy

## Supported version

The latest commit on the `main` branch is the supported version of this research repository.

## Reporting a vulnerability

Please do not disclose security-sensitive issues publicly before they have been reviewed. Report a suspected vulnerability through GitHub's private vulnerability reporting feature when it is available for this repository, or contact the repository owner through the GitHub profile linked from the project.

Include:

- the affected file or component;
- steps to reproduce the issue;
- the potential impact;
- any proposed mitigation.

You should receive an acknowledgement within seven days. Research-correctness issues, reproducibility problems, and ordinary software defects may be reported through GitHub Issues.

## Scope

This repository processes a separately downloaded public research dataset. Do not commit credentials, private datasets, personal information, trained models containing sensitive data, or untrusted serialized artifacts.
