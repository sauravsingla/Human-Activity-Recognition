# Results

This directory is reserved for small, reviewable result summaries generated from complete experiments.

Large trained models, probability arrays, datasets, and other generated artifacts are ignored by Git and should not be committed. The default training output directory is `artifacts/`.

## Reporting policy

A result should be added here only when all of the following are recorded:

1. model and configuration (`paper` or `modern`)
2. dataset split and sample counts
3. random seed
4. TensorFlow, NumPy, Python, and platform versions
5. epochs requested and completed
6. batch size and model parameter count
7. accuracy, macro precision, macro recall, macro F1, and confusion matrix
8. the generated `metrics.json` or three-model comparison summary

Historical values visible in the original notebooks or publication must be labelled **reported in the paper**. Values generated by the current package must be labelled **reproduction run**. They should not be mixed.

Run the full comparison with:

```bash
python -m har.compare --dataset-dir "/path/to/UCI HAR Dataset" --configuration paper
```
